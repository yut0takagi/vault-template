# temp-folder プロジェクト設定

汎用作業ディレクトリ。コード開発、リサーチ、プロトタイピング、一時的な作業に使用。

## ルール

- 日本語で応答する
- 成果物はこのディレクトリ内に作成する
- 不要になったファイルは都度削除して整理を保つ

## 完了宣言プロトコル

「完了しました」等の完了表現は、以下を通過した後にのみ使用可。

1. **元の依頼を再読** — 要求を全て満たしたか確認
2. **成果物の実在確認** — ファイルが存在し内容が期待通りか
3. **タイプ別検証:**
   - コード → ビルド/lint/テスト実行
   - ドキュメント → 必須セクション確認
4. **差分確認** — 依頼外の変更が含まれていないか
