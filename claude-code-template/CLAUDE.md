## ルール

### 応答スタイル

- 日本語で応答すること
- 簡潔に要点を伝える。冗長な説明は不要
- コード変更時は変更理由を一言添える

### コーディング規約

- 既存コードのスタイルに合わせる
- 不要なコメントやドキュメントを追加しない
- 依頼された範囲外のリファクタリングをしない

### Git

- コミットメッセージは日本語で書く
- コミットは明示的に依頼されたときのみ行う
