# タグ定義

ナレッジで使用するタグの一覧と運用ルール。タグの乱立を防ぐため、新規タグ追加時はここを確認する。

---

## 構造タグ（全ファイル共通）

| タグ | 用途 |
|------|------|
| `ナレッジ` | ナレッジファイル共通 |
| `議事録` | 議事録ファイル共通 |
| `日報` | 日報ファイル共通 |

---

## カテゴリタグ（フォルダと対応）

| タグ | 対象 |
|------|------|
| `技術` | 技術・開発系ナレッジ |
| `ツール` | ツール・サービス系 |
| `リサーチ` | 調査・分析結果 |

---

## 内容タグ

<!-- プロジェクトに合わせてここにタグを追加する -->

| タグ | 意味 |
|------|------|
| `AI活用` | AI全般の活用 |
| `設計` | アーキテクチャ・設計関連 |
| `業務効率化` | 業務自動化・効率化 |

---

## 運用ルール

1. **構造タグ + カテゴリタグ + 内容タグ3〜5個** を目安にする
2. 1回しか使わないタグは作らない（既存タグで代替）
3. 新規タグを追加する前にこのファイルを確認する
4. プロジェクト固有名はタグにせず、本文内で検索する
